# RAG UI Icon Replacements

替换截图里偏硬的 `R`、`问`、`我 / R` 字符头像。

- `brand-mark.svg`：左上角主品牌图标，墨绿底 + 书页 + 星点。
- `ask-seal.svg`：朱砂副标，保留印章感但不用生硬文字。
- `user-avatar.svg`：用户头像，宣纸底 + 墨绿线稿。
- `assistant-avatar.svg`：助手头像，墨绿底 + 书页符号。

建议尺寸：

- 左上角品牌主标：48 x 48
- 左上角朱砂副标：40 x 40
- 导航用户头像：36 x 36
- 消息流头像：36 x 36

视觉建议：

- 主标和助手头像使用同一个“书页 + 星点”母题，建立识别。
- 用户头像不要再写“我”，用人物线稿更自然。
- 朱砂副标只当点缀，避免和主标抢识别。
